# Programming Concepts

### 阻塞/非阻塞 同步/异步
阻塞：线程在执行操作时不能执行其他操作。  
非阻塞：线程在执行操作时可执行其他操作。  
同步：调用等待返回结果，不可执行其他操作。  
异步：调用者直接返回，可执行其他操作。  

### 面向对象/函数式
```
面向对象编程(OO)通过封装变化使得代码更易理解。
函数式编程(FP)通过最小化变化使得代码更易理解。
          —— Michael Feathers(Twitter)
```

### 纯函数
1. 仅取决于提供的输入，而不依赖于任何在函数求值期间或调用间隔时可能变化的隐藏状态和外部状态。
2. 不会造成超出其作用域的变化，例如修改全局对象或引用传递的参数。
